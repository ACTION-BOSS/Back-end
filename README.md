# 실전프로젝트 - 행동대장
동네 사건,사고와 문제에 보다 쉽게 접근하고 공유하는 커뮤니티 서비스

---

### 팀원
김예진, 이예원, 최순

---

### 결과물

홈페이지 주소 : [행동대장](https://hdaejang.com/)

커뮤니케이션: [jira: 행동대장](https://the-strongest-team4.atlassian.net/wiki/spaces/BOSS/overview?homepageId=2720123)

---

### 주요 기능
동네사람들과 불편사항 공유
 - 불편사항 게시글 작성, 수정
 - ‘나도 불편해요’ 표시로 공감
 - ‘해결했어요’ 표시로 불편사항이 해결됨을 공유
 - 카카오 지도 API
 - 지도와 핑으로 직관적인 위치 확인
